Dating App

Bienvenido al modo demo. Esta vista estática permite comprobar que el proyecto puede servir contenido incluso si npm install está bloqueado por el proxy.

¿Por qué esto funciona?

  • Usa solo dependencias nativas de Node.js.
  • No requiere descargar paquetes externos.
  • Se ejecuta con npm run demo.

¿Qué sigue?

  1. Arranca el servidor demo con npm run demo.
  2. Abre http://localhost:3000 en tu navegador.
  3. Cuando el proxy permita salidas al registro de npm, ejecuta el script de instalación habitual para volver al modo completo.

Si ya puedes instalar dependencias, detén este servidor y usa npm run dev como de costumbre.
